Tale 博客系统

Tale 的英文含义为故事，我相信每个坚持写 Blog 的人都是有故事的；中文你叫它 塌了 也无所谓 🤣。

  Tale 使用了轻量级 mvc 框架 Blade 开发，默认主题使用了漂亮的 pinghsu，如果觉得这个项目不错，请为它点赞支持。
演示站点：https://tale.biezhi.me

安装JDK

其实安装jre就行，大部分时间都用不到JDK，在终端输入下面命令更新系统再安装jre：

sudo apt-get update
sudo apt-get instail default-jre

java -version

安装Tale

通过FTP工具安装（待写…）

后台运行Tale

在终端输入下面命令：

nohup java -Xms128m -Xmx128m -Dfile.encoding=UTF-8 -jar tale-least.jar >/dev/null &amp;

默认9000端口
