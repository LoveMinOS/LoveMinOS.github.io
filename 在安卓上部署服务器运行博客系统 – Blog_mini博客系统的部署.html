Blog_mini是什么？

  Blog_mini是一个用Python开发的，拥有简洁页面（支持响应式布局！）和强大后台管理功能的开源博客系统，使用Blog_mini，你将能轻松使用和管理你的个人博客。

强大管理后台：

说明

  本文只对 Blog_mini博客系统的部署进行简单的介绍，通过本文的阅读，你将可以在一个全新的Ubuntu系统上完全部署Blog_mini，从而成功架设属于你的个人博客网站。
  这里使用的操作系统为：Ubuntu 15.10，理论上，只要操作系统上安装的Python版本为2.6.x或2.7.x，都可以成功部署Blog_mini。

获取Blog_mini源代码

  有以下两种方式可以获取Blog_mini的源代码：

  • 通过git的方式
  • 通过Blog_mini项目主页下载
      在这里我将对这两种方法进行介绍

    这里推荐第一种方法。

通过git获取源代码

先要确定系统已经安装了git版本控制系统，在终端输入下面命令：

git version

  • 如果安装了，终端则会显示git的版本信息，如果“git”没有安装，可以在终端输入下面命令通过apt-get方式安装：

    sudo apt-get install git

    无法安装的话可以切换到超级用户（root）下在终端输入上面命令。

  • 安装完成后，切换到普通用户，在普通用户用户目录中创建一个文件夹用于存放Blog_mini文件，这里我根据作者部署文档创建一个名为project的目录：

    mkdir project
cd project

    在project目录下在终端输入下面命令从Blog_mini项目地址中克隆源代码：

    git clone https://github.com/xpleaf/Blog_mini

    如果网络情况良好的话，很快就可以把代码克隆下来。

  • 现在在project目录下就出现了一个名为Blog_mini的文件夹；进入Blog_mini目录下，可以查看Blog_mini的源代码目录结构：

    cd Blog_mini
ls

    通过Blog_mini项目主页下载源代码

      如果你想通过该方式获得Blog_mini的源码的，那么请最好在有GUI界面的操作系统中进行操作，比如你可以先在Windows或Mac或有GUI界面的Linux操作系统中下载下来，再把它传到我们的Ubuntu操作系统中。

  • 进入项目主页：
    https://github.com/xpleaf/Blog_mini

  • 点击Download ZIP按钮
  • 下载后的文件名应该是：Blog_mini-master.zip

下载完成后，将其解压缩并通过fpt传到Ubuntu操作系统中即可，Blog_mini的源码目录结构与之前的通过git的方式是一样的。

安装python和pip

  在Blog_mini目录下输入下面命令：

sudo apt-get install python
sudo apt-get install python-pip

安装virtualenv

  在Blog_mini目录下输入下面命令：

sudo apt-get install virtualenv

  如果上面的命令说找不到virtualenv的，就使用下面的命令：

sudo apt-get install python-virtualenv

创建虚拟环境venv

  在Blog_mini目录下输入下面命令：

virtualenv venv

激活虚拟环境venv

  在Blog_mini目录下输入下面命令：

source venv/bin/activate

注意看提示符的改变。

安装Blog_mini需求文件

  • 执行如下命令：
    pip install -r requirements/common.txt
      因为这里要安装Blog_mini所需的一些扩展包，所以需要一定时间，当然如果网络良好的话，很快就可以安装完成。
